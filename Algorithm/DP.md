# 동적계획법 (DP; Dynamic Programming)

- 최적화 이론의 한 기술이며, 특정 범위까지의 값을 구하기 위해 그것과 다른 범위까지의 값을 이용하여 효율적으로 값을 구하는 알고리즘

- 그리디 알고리즘과 같이 최적화 문제를 해결하는 알고리즘

- 예시

    ```Python
    # 피보나치 수 DP적용 알고리즘

    def fibo(n):
            f = [0] * (n+1)
            f[0] = 0
            f[1] = 1
            for i in range(2, n+1):
                f[i] = f[i-1] + f[i-2]
            return f[n]
    ```
